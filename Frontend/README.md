# qaPI Frontend

React frontend for SentinelTwin with beautiful dark neon gradient theme

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
